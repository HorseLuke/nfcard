# 中文

该库是由@sinpo 在Google Code上建立的同名NFCard项目（安卓手机应用）复制而来。源代码见：https://code.google.com/p/nfcard/

NFCard使用手机NFC读取非接触式IC卡中的信息。 支持ISO7816-4、Felica智能卡，用来读取电子钱包（主要是公交卡）中没有设置读取限制的余额、交易记录。
